# Building Height and Footprint

Adds real-world building measurements to the Building Info panel in *Cities: Skylines II*.

When selecting a building, this mod displays its estimated height, floor count, and ground footprint alongside the game’s existing information.

---

## Features

- **Building height**
  - Displayed in feet or meters
  - Derived from actual 3D building geometry

- **Estimated floors**
  - Calculated using realistic floor-height heuristics
  - Adapts to low-rise, mid-rise, and high-rise buildings

- **Building footprint**
  - Estimated ground area from geometry
  - Displayed as:
    - Acres (Imperial)
    - Hectares (Metric)
  - Approximate zoning cell dimensions for zoned buildings

- **Building elevation**
    - Displayed in feet or meters
    - Adjustable offset in case you don't want to use the map's built in sea level value.

- **UI integration**
  - Zoned buildings: Level section
  - Ploppable & signature buildings: Description section

---

## Settings

Available under **Options → Building Height and Footprint**:

- Height units: Feet or Meters  
  (Footprint units switch automatically between acres and hectares)
- Elevation offset: Adjust the displayed elevation by a fixed amount (in meters)

---

## Notes

- Informational only — no gameplay or simulation changes
- Works with vanilla and modded buildings
- Values are estimates based on available geometry data
